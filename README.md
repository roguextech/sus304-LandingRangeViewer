# LandingRangeViewer
学生ロケット界隈でよく使用されている風向風速による着地/着水範囲の可視化ツール

This software is released under the MIT License.  
本ソフトウェアはMITライセンスのもとで公開されている。

## 使い方
1. 着地地点を(X,Y)=(磁東,磁北)で記述したcsvファイルを作成  
フォーマットはサンプルを参考に
2. Openボタンで上記のcsvファイルを選択
3. 条件を選択し、画像を保存

## 対応状況
* 1-7 m/sの風速  
1-7がないと動作しない
* 360degを45deg分割の風向
* 能代陸、能代海に対応  
伊豆大島など対応予定
